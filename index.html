About

I'm a GIS and urban planning professional with seven years of experience using geospatial technology and data analysis to develop innovative, information-driven ways to make better historic preservation and planning decisions. Areas of expertise include both Esri and open-source GIS software, advanced geospatial analysis, data creation, management and maintenance, database design, geospatial project management and custom cartography. In addition, I'm also experienced in the development of custom web mapping applications using the Esri ArcGIS Javascript API and MapBox API as well as open-source options like Leaflet.
