Exploring Neighborhood Change

Description:Created as part of the National Trust's report, Preserving African American Places: Growing Preservation's Potential as a Path for Equity, the Exploring Neighborhood Change map series focuses on the impact of equitable development, social justice and historic preservation in 10 cities across the US. Additional applications for the other 9 study cities can be found here: Atlanta, Birmingham, Chicago, LA, Louisville, Oakland, Philly, St. Louis & DC. Project data created by National Trust staff. Popup charts created using Chart.js.

Project Type:Professional

Organization:National Trust for Historic Preservation

Technology Used:ArcMap 10.4, ArcGIS Pro 2.6, QGIS 3.14, PostgreSQL, ArcGIS API for Javascript 4.x, Chart.js

Historic Markers of Virginia

Description:The Historic Markers of Virginia app was created out of a desire to see where the state’s numerous markers are located and if mapping their location could help people understand Virginia's history better. I hope the answer is yes. Historic Marker location data was obtained from the Virginia DHR but cleaning, formatting and categorization was completed by me. All icons were made by me using Adobe Illustrator. Map tiles by Stamen Design.

Project Type:Personal

Technology Used:ArcMap 10.8, QGIS, Adobe Illustrator (Icons), ArcGIS API for Javascript 4.x

Visualizing Covid-19

Description:The Covid-19 pandemic has had an unimaginable impact on the US and the world. In October 2020 I became interested in portraying the pandemic at the county-level in a 3D environment. I obtained Covid data from the New York Time’s public GitHub data repository, cleaned and formatted it and then created different visualizations of the pandemic’s progress across the US from February through October 2020. Popups contain Chart.js charts that show infection and death rates.

Project Type:Personal

Technology Used:ArcMap 10.8, R Studio, ArcGIS API for Javascript 4.x

Mapping the Blue Ridge Parkway

Description:This application was created because of a lack of mapping tools that portrayed, in detail, all 469 miles of the Blue Ridge Parkway. This application seeks to do that and contains detailed information on all parkway overlooks, views, visitor centers, lodging options, and points of interest. The app also portrays parkway tunnels and some bridges. Click on hiking trails to get a difficulty rating and distance. All data was created by me except for the parkway boundary. Icons courtesy of the National Park Service. All 3D models including bridges and tunnel faces were created with AssetForge.

Project Type:Personal

Technology Used:Arcmap 10.8, ArcGIS Pro 2.6, AssetForge 2.1.1, ArcGIS API for Javascript 4.x

Historic Falmouth, Jamaica

Description:I spent the summer of 2012 working in historic Falmouth, Jamaica and always wanted to see the town portrayed in a 3D mapping application. Some geospatial data was available from the University of Virginia’s Falmouth Project, which has done extensive documentation work in the town, and I created the vectortile basemap from a 1959 map completed by the Survey Department of Jamaica. All 3D buildings were completed by me in Esri’s CityEngine. Building forms were started using CGA rules and then 18th and 19th century historic buildings were given additional details manually. All popup images and building descriptions courtesy of the Falmouth Project.

Project Type:Personal

Technology Used:ArcMap 10.8, QGIS 3.14, ArcGIS Pro 2.6, CityEngine 2020, ArcGIS API for Javascript 4.x

Historic Tax Credit Impacts

Description:Since 1976, the Federal Historic Tax Incentives program has been a key part of encouraging private sector investment in the rehabilitation of historic buildings. In the 44 years that the program has existed more than $102 billion dollars of private investment has been utilized to preserve over 45,000 buildings. This application portrays historic tax credit use at the state, congressional district, and local level in terms of frequency, economic impact and project type. Data from the National Park Service - cleaned, organized and formatted by me - and charts created using Chart.js. Maptiles by Carto.

Project Type:Personal

Technology Used:ArcGIS 10.8, R Studio, ArcGIS API for Javascript 4.x, Chart.js

Visit Drayton Hall

Description:This application was completed as a demonstration of how 3D mapping technology could be applied to historic sites that have been closed due to the pandemic and are in need of digital tools that can provide a virtual tour experience. This application showcases Drayton Hall, located on the Ashley River outside of Charleston, South Carolina and was completed using a combination of Trimble SketchUp, Esri CityEngine and ArcGIS Pro. The scale model of the main house at Drayton Hall was created using schematics created by the National Park Service’s Historic American Building Survey and are publicly available at the Library of Congress.

Project Type:Personal

Technology Used:ArcGIS 10.8, ArcGIS Pro 2.6, CityEngine 2020, Sketchup, Blender, ArcGIS API for Javascript 4.x

Landmarks Permit Finder

Description:Completed in 2018 for the New York City Landmarks Preservation Commission, the Permit Finder application enables members of the public to quickly and efficiently find out information on issued and pending building permits for properties protected by the city’s landmarks preservation law. This application was featured in the plenary session of Esri’s 2020 User Conference and will also be included in Esri’s Map Book Volume 36.

Project Type:Professional

Organization:New York City Landmarks Preservation Commission

Technology Used:ArcMap 10.4, ArcGIS Online, ArcPy, ArcGIS WebAppBuilder

Landmarks Data Development

Description:Begun in 2017, this ground-breaking data creation project resulted in a historic building database that contains information on all 36,000+ designated buildings in New York City. I developed workflows for the creation of spatial data from pdf documents of Landmarks Preservation Commission (LPC) designation reports and led a team of four GIS analysts as the data was painstakingly assembled. The data is now publicly available through the NYC Open Data Portal and is consistently updated by current LPC staff.

Project Type:Professional

Organization:New York City Landmarks Preservation Commission

Technology Used:ArcMap 10.4
